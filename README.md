# some_project

## Native application

### Install
```
npm install
cd ios
pod install
```

### Run
```
// in react native root dir
// if you want to run android
npm run android

// if you want to run ios
npm run ios
```

## Server(Deprecated)

Need to install Nest.js first
`npm i -g @nestjs/cli`

### Install
```
// in root dir
npm install
```

### Run
```
// in server dir
npm run start
```
